# Web Profile Santri - Django

Project ini adalah website profile santri yang dibangun menggunakan framework Django.

## Fitur
- Homepage Portfolio
- Profil Santri (Biodata, Jadwal, Galeri)
- Template Inheritance (Base Template)
- Custom CSS dengan Font Poppins

## Cara Menjalankan Project
1. Pastikan Python terinstall
2. Jalankan Virtual Environment: `.venv\Scripts\activate` (Windows)
3. Install Django: `pip install django`
4. Jalankan Server: `python manage.py runserver`
